# Identificación de Actores

## 1. Agricultor (Actor Principal)

Es el usuario principal del sistema y quien realiza la mayoría de las operaciones relacionadas con la gestión agrícola.

### Casos de Uso

#### Gestión de Cuenta

* Iniciar sesión.
* Cerrar sesión.
* Actualizar perfil.

#### Gestión de Parcelas

* Registrar parcela.
* Consultar parcelas.
* Actualizar parcela.
* Eliminar parcela.

#### Gestión de Cultivos

* Registrar cultivo.
* Consultar cultivos.
* Actualizar cultivo.
* Eliminar cultivo.

#### Gestión de Actividades Agrícolas

* Registrar riego.
* Registrar fertilización.
* Registrar aplicación de pesticidas.
* Registrar poda.
* Registrar cosecha.
* Consultar historial de actividades.

#### Monitoreo de Cultivos

* Registrar observaciones.
* Registrar fotografías.
* Consultar estado del cultivo.
* Consultar historial de incidencias.

#### Gestión de Inventario

* Registrar producto.
* Actualizar stock.
* Consultar inventario.
* Eliminar producto.

#### Gestión Económica

* Registrar gasto.
* Registrar ingreso.
* Consultar gastos.
* Consultar ingresos.
* Consultar rentabilidad.

#### Reportes y Estadísticas

* Consultar reporte de producción.
* Consultar reporte de costos.
* Consultar rendimiento por parcela.
* Consultar estadísticas agrícolas.

#### Alertas

* Consultar alertas.
* Marcar alerta como atendida.

#### Sincronización

* Sincronizar datos con el servidor.

#### Inteligencia Artificial

* Solicitar análisis de enfermedad mediante fotografía.
* Consultar recomendaciones de riego.
* Consultar predicción de rendimiento de cosecha.

---

# 2. Técnico Agrícola (Actor Secundario)

Participa en el monitoreo y asesoramiento técnico de los cultivos.

### Casos de Uso

#### Inspecciones

* Registrar inspección agrícola.
* Consultar inspecciones realizadas.

#### Incidencias

* Registrar incidencia.
* Consultar incidencias registradas.

#### Recomendaciones

* Registrar recomendación técnica.
* Consultar recomendaciones emitidas.

#### Evidencias

* Adjuntar fotografías.
* Registrar observaciones técnicas.

#### Monitoreo

* Consultar estado de cultivos.
* Consultar historial de actividades agrícolas.

---

# 3. Servicio Meteorológico (Actor Externo)

Sistema externo encargado de proporcionar información climática.

### Casos de Uso

#### Información Climática

* Proporcionar pronóstico meteorológico.
* Proporcionar temperatura actual.
* Proporcionar humedad ambiental.
* Proporcionar probabilidad de lluvia.
* Proporcionar velocidad del viento.

### Interacción con el Sistema

Agronova consulta periódicamente:

* Pronóstico climático.
* Condiciones actuales.
* Alertas meteorológicas.

Esta información alimenta:

* Alertas.
* Recomendaciones de riego.
* Predicciones agrícolas.

---

# 4. Servicio de Notificaciones (Actor Externo)

Sistema encargado del envío de notificaciones automáticas a los usuarios.

### Casos de Uso

#### Gestión de Notificaciones

* Enviar alerta de riego.
* Enviar alerta de fertilización.
* Enviar alerta de cosecha.
* Enviar alerta de stock bajo.
* Enviar recordatorios automáticos.

### Interacción con el Sistema

Agronova genera la alerta y el servicio externo la distribuye al dispositivo móvil.

---

# 5. Motor de Inteligencia Artificial (Actor Externo)

Sistema especializado en análisis predictivo y procesamiento de imágenes.

### Casos de Uso

#### Diagnóstico Agrícola

* Analizar fotografía de cultivo.
* Detectar enfermedades.
* Identificar posibles plagas.

#### Predicción

* Predecir rendimiento de cosecha.
* Analizar datos históricos.

#### Recomendaciones

* Recomendar frecuencia de riego.
* Recomendar acciones preventivas.
* Recomendar tratamientos agrícolas.
